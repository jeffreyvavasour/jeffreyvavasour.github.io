# Darts App

This is a two player darts scorer application. Users can set the score limits to 501 or 301. It will provide previous scores hit and also possible checkouts when reaching certain scores. It will not allow impossible scores, impossible checkouts or empty inputs. Users can also choose how many sets and legs they wish to play for. After the match has finished, users will have the options to play again, go to the home screen, or look up the match statistics.
